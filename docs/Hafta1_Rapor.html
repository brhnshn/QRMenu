2. Solution ve Proje Oluşturma

> dotnet new sln -n QRMenu
"Çözüm Dosyası" şablonu başarıyla oluşturuldu.

> dotnet new classlib -n QRMenu.Core -o src\QRMenu.Core --framework net8.0
> dotnet new classlib -n QRMenu.Data -o src\QRMenu.Data --framework net8.0
> dotnet new mvc -n QRMenu.Web -o src\QRMenu.Web --framework net8.0
> dotnet new xunit -n QRMenu.Tests -o tests\QRMenu.Tests
Tüm projeler başarıyla oluşturuldu. ✅

4. Veritabanı Entity'leri (13 Tablo)

Entity Açıklama Önemli Alanlar
Masa Restoran masaları MasaNo, AktifMi, QrKodUrl
Kategori Ürün kategorileri Ad, AdEN, SiraNo, AktifMi
Urun Menü ürünleri Ad, Fiyat, GorselUrl, PopulerMi, SatisSayisi
Opsiyon Ürün seçenekleri Ad, Grup (Boyut/Süt), EkFiyat
UrunOpsiyon M2M ilişki tablosu Composite Key: UrunId + OpsiyonId
Oturum Token bazlı oturum TokenHash, Kullanildi, SonIslemTarihi
Sepet Müşteri sepeti OturumId, ToplamTutar
SepetDetay Sepetteki ürünler UrunId, Adet, SeciliOpsiyonlar
Siparis Siparişler Durum (9 enum), ToplamTutar, RowVersion
SiparisDetay Sipariş kalemleri UrunId, Adet, BirimFiyat
Odeme Ödeme kayıtları Tutar, OdemeTipi (Nakit/Kredi)
Kullanici Personel KullaniciAdi, Rol (4 rol enum)
AuditLog Değişiklik logları TabloAdi, Islem, EskiDeger, YeniDeger

6. Seed Data

  • 5 Masa — Masa 1-5 (AktifMi = true)
  • 3 Kategori — Sıcak İçecekler, Soğuk İçecekler, Atıştırmalıklar
  • 10 Ürün — Filtre Kahve, Latte, Americano, Türk Kahvesi, Buzlu Latte, Limonata, Smoothie, Cheesecake, Brownie, Sandviç
  • 5 Opsiyon — Küçük, Orta, Büyük, Tam Yağlı Süt, Yağsız Süt
  • 16 UrunOpsiyon — Filtre Kahve, Latte, Americano, Buzlu Latte için boyut + süt seçenekleri

7. Migration & Supabase

> dotnet ef migrations add InitialCreate --project src\QRMenu.Data --startup-project src\QRMenu.Web
Migration 'InitialCreate' oluşturuldu.

> dotnet ef database update --project src\QRMenu.Data --startup-project src\QRMenu.Web
Veritabanı güncellendi. Tüm tablolar Supabase PostgreSQL'de oluşturuldu. ✅
