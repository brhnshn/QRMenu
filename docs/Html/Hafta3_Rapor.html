3. Uygulanan UX Anatomisi (Mobil Web App Tasarımı)

Sade ve yüksek dönüşüm odaklı tasarım mimarisi:
  • Tepe Alanı: Logo + Masa Numarası doğrulaması (Masa 1). Sağ üstte TR/EN.
  • Off-canvas Kategori: "Kategoriler" butonuyla açılan yan menü navigasyonu.
  • Ürün Kartları: Yatay (Horizontal) liste düzeni. Kare fotoğraf solda, açıklama ve fiyat sağda. Okunabilirlik %100.
  • Sticky Bottom Bar: Ekranın en altında sabit duran "Garson" ve "Sepeti Gör" aksiyon çubuğu.
  • Off-canvas Sepet: Sayfa yenilemeden sağdan kayan sepet penceresi.

4. Teknik Detaylar & Performans Optimizasyonu

4.1 AJAX Bazlı Sepet Akışı

// JavaScript (AJAX) ile ürün ekleme örneği
async function postAddToCart(urunId, adet, opsiyonIds) {
    const response = await fetch('/sepet/ekle', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ UrunId: urunId, Adet: adet, OpsiyonIds: opsiyonIds })
    });
    const data = await response.json();
    if (data.success) updateBottomBar(data.urunSayisi, data.toplamTutar);
}

4.2 Latency Öneleyici (Tek Transaction)

Stockholm - Türkiye arasındaki ping süresini ekarte etmek için 4 ayrı veritabanı çağrısı yerine, sepet hesaplaması bellekte (memory) yapılıp veritabanına tek bir SaveChangesAsync() ile gönderildi. Yazılım hızı 4 katına çıkarıldı.
